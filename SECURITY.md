# Security Policy

## Supported Versions

This project is maintained on the latest `main` branch. Security fixes are applied to the latest version only.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

- Prefer GitHub Security Advisories (private disclosure):  
  https://github.com/JoyalJJoy/AI-Doc-Search/security/advisories

If you cannot use GitHub advisories, open a minimal issue and request a private follow-up from the maintainer.

We aim to acknowledge reports within 7 days and provide a fix or mitigation timeline as soon as possible.
